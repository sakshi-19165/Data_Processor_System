# Automated Data Entry & Processing System

A Python-based automated data processing pipeline that ingests CSV/Excel files,
cleans and validates data, and stores structured records in MongoDB Atlas.

## Tech Stack
- Python
- Pandas
- MongoDB Atlas
- PyMongo

## Features
- CSV and Excel file ingestion
- Data cleaning and validation
- Secure MongoDB Atlas integration
- Modular and scalable project structure
